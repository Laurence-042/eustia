# Abbreviation

## Always Use Abbreviation

|Original|Abbreviation|
|--------|------------|
|argument|arg|
|array|arr|
|attribute|attr|
|module|mod|
|callback|cb|
|constructor|ctor|
|context|ctx|
|element|el|
|function|fn|
|index|idx|
|number|num|
|object|obj|
|return|ret|
|string|str|
|template|tpl|
|value|val|

## Use Abbreviation When Compound

|Original|Abbreviation|
|--------|------------|
|command|cmd|
|current|cur|
|delete|del|
|origin|orig|
|remove|rm|
